## Usage

Run the provided script to download all FASTQ files:

```bash
bash download_sra.sh
```

The script uses `wget -c` (continue/resume) and downloads both read 1 (`_1.fastq.gz`) and read 2 (`_2.fastq.gz`) for paired-end samples from the [EBI FTP mirror](https://www.ebi.ac.uk/ena/browser/).
